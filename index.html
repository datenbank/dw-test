Welcome to DW Test Toolkit.

The Data Warehouse Test Toolkit is made to assists you when testing a Data Warehouse , and ensuring the quality. The toolkit lets you write queries for source and target systems and compare the results for you and report any differences.

Download/install

Go to https://github.com/datenbank/dw-test and download the latest version, and extract the zip file to your prefered destination.

Run

Navigate to the bin folder, and type dw-test.bat --gui to open the toolkit graphical user interface.

To setup connections to source and target databases right click anywhere in the table view. select Settings->Open file. Edit the connection string to match your requirements. Save your settings by clicking Save button, and close the window. Through the right click menu click Settings->Reload to load in the new settings. Settings are read on startup, and when Reload menu item is clicked.

Windows Authentication

To use Windows Authentication with MS SQL Server leave out user and password from connection string and download jTDS from https://sourceforge.net/projects/jtds/ extract the zip file and put the for example C:\jtds-1.3.1-dist\x64\SSO in PATH (environment variable)
