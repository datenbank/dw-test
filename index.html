Welcome to DW Test Toolkit.

The Data Warehouse Test Toolkit is made to assists you when testing a Data Warehouse , and ensuring the quality. The toolkit lets you write queries for source and target systems and compare the results for you and report any differences.

Download/install

Go to https://github.com/datenbank/dw-test and download the latest version, and extract the zip file to your prefered destination.

Run

navigate to the bin folder, and type dw-test.bat --gui to open the toolkit graphical user interface.
